TOVISEN Australia - GitHub Pages Only

Upload BOTH files in this folder directly into the root of a new GitHub repository:

- index.html
- .nojekyll

Then:
1. Open the repository's Settings.
2. Select Pages.
3. Under Build and deployment, choose Deploy from a branch.
4. Choose main and /(root).
5. Click Save.

Your public website URL will normally be:
https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/

Important:
- Use a public repository if you are using GitHub Free.
- Do not upload the confidential internal market-strategy presentation.
- Internet access is required for the machine images and YouTube videos.
